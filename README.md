## Setup Instructions

1. **Extract the portfolio folder**
2. **Add your profile picture**: 
   - Name it `profile.jpg`
   - Place it in the main portfolio folder
   - Recommended size: 500x500px square
3. **Open `index.html`** in your web browser
4. **Customize**: Edit the HTML to update your information

## Customization

### Update Personal Information
In `index.html`, modify:
- Name and contact details
- Education history
- Projects and descriptions
- Skills percentages

### Change Colors
In `CSS/portfolio.css`, edit the `:root` section:
```css
--primary: #00f7ff;    /* Main color */
--secondary: #6c63ff;   /* Accent color */
```

### Contact Form
The form currently shows an alert. To make it functional:
- Use **EmailJS** (free service)
- Use **Formspree** (simple form backend)
- Create your own backend with PHP/Node.js
